# Task 4: Variáveis de Ambiente

❌ **Pendente** — adicionar ao `.env`

Adicionar ao arquivo `.env`:
- `JIRA_BASE_URL` — URL base da instância Jira (ex: `https://sua-empresa.atlassian.net`)
- `JIRA_EMAIL` — email do usuário Jira com permissão de criar issues
- `JIRA_API_TOKEN` — token de API do Jira
- `JIRA_PROJECT_KEY` — chave do projeto (ex: `ZSCAN`)
